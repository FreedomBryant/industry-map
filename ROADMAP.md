# 中国产业地图 — 后续规划

> 当前阶段：V1 已实现
> - 中国省级热力地图（34省）
> - 鼠标悬浮提示
> - 点击省份展开详情面板（饼图 + 柱状图 + 企业 + 城市列表）
> - 80+ 城市级产业数据

## 📌 后续可做功能（按优先级）

### P1 — 核心交互增强

- [x] **产业筛选器**：按产业分类（IT、制造、能源、农业、金融、文旅、生物医药等）高亮地图，只显示选中产业的省份
- [x] **省份对比**：多选 2~4 个省份，并排对比产业分布饼图/产值柱状图
- [x] **搜索框**：输入省份或城市名，地图自动定位+高亮

### P2 — 数据与分析

- [x] **时间轴**：选择年份（2020~2024），查看产业变迁趋势
- [x] **产业链图谱**：点击某个产业（如"新能源汽车"），展示全国上下游分布
- [x] **数据导出**：省份详情导出为图片/PDF
- [x] **数据源说明面板**：显示数据口径、来源、更新时间

### P3 — 体验与视觉

- [x] **深色模式**：适配 dark theme
- [ ] **移动端适配**：地图缩放 + 面板滑动抽屉
- [ ] **飞线动效**：展示跨省产业关联（如总部-工厂流向）
- [ ] **响应式仪表盘**：顶部展示全国产业汇总指标（GDP 总和、企业数量等）

### P4 — 扩展数据

- [ ] **区县级下钻**：点击城市展开区县
- [ ] **企业级数据**：搜索具体企业，在地图上标注全国分布
- [ ] **园区/开发区**：标注国家级高新区、经开区
- [ ] **贸易流向**：省份间进出品贸易关联

## 🔧 技术备忘

| 项目 | 当前方案 | 优化方向 |
|------|----------|----------|
| 地图 | ECharts + GeoJSON（DataV） | 对接 Mapbox / 高德地图 API |
| 数据 | 手写静态 TS 数据 | 改为 JSON 文件动态加载，便于替换 |
| 状态管理 | 组件内 props + emits | 引入 Pinia 管理跨组件状态（筛选、年份） |
| 构建 | Vite 6 | 保持 |
| 图表 | ECharts 5 | 保持 |
